# 🧠 Mindcare — Skincare for the Mind

A calm, minimal landing page designed to feel like *mental skincare* — structured, intentional, and timeless.

This project translates a wellness-focused brand system into a clean, responsive web experience using **HTML, CSS, and vanilla JavaScript**.

---

## 🌿 Overview

Mindcare is a concept for a mental wellness product that helps users pause, reflect, and reset.

The design philosophy is inspired by:

* Simplicity over noise
* Calm over urgency
* Intention over excess

---

## 🎨 Design Principles

* **Calm:** Soft, neutral colour palette with low contrast
* **Structured:** Clear layout hierarchy and spacing system
* **Intentional:** Minimal UI with focused interactions
* **Timeless:** No trendy elements — built to last

---

## 🧩 Features

* Responsive landing page (mobile + desktop)
* Clean, minimal UI components
* Subtle scroll-based animations (Intersection Observer)
* Reusable design system (buttons, cards, layout)
* Lightweight — no frameworks or libraries required
